    There is a new app for the iPhone and iPad called iReality. This app is in the testing stage so only a few people actually have it.

    iReality scans everything within either 20 feet, 40 feet, 80 feet, or a whole building when it's turned on. The scan distance can be changed through the settings menu (the settings menu is automatically entered the first time iReality is run and can be accessed later by taking two fingers and swiping from left to the right). The option to scan a building is not available if the user is outside of a building and the app will scan within 80 feet instead.

    No matter what scan range is selected an overhead view of the surrounding area will appear on the screen. To zoom in the user takes two fingers and spreads them on the screen and to zoom out the user pinches their fingers together. Tapping on something highlights it then the user takes two fingers and swipes from right to left to bring up the options for that object or person. Some of the options include name, species, age, sex, size, nationality, color, material, animate/inanimate, sexual orientation, aware of the reality shift, and level of awareness. There are many more options available to the user. There are several windows for the options and to change the window the user takes a finger and swipes it either to the left or the right.

    When a building is scanned iReality will display an overhead view of the floor the user is on. To change floors the user uses two fingers and swipes from top to bottom to go up a level and from bottom to top to go down a level. On the top and bottom levels is an option to add a level. When you're in a building you can reassign rooms and move objects from one room to another or into the app's inventory. The inventory is only limited by memory and only objects can be placed inside it (people turned into objects can be placed inside the inventory). To access the inventory the user taps on the screen two times with two fingers and then they tap on an object once to bring up the option to move it out of the inventory and to a room or to delete it (people turned into objects can not be deleted).

    When the user has finished editing the options the way they want them they tap the Save button and reality instantly shifts. Reality will generally change only as much as needed to meet the edited options.

    In the settings menu the user can lock iReality to themselves to prevent anyone else from using the app. If the user changes themselves into an object a safety feature activates and will revert the change in as little as one hour or as long as one week depending on what the user sets it as (it is advised that the user uses extreme caution if they change themselves into an object while the app is locked to them as it is in testing and the safety feature may suddenly stop working). The user can also set their awareness to always being aware of the shift and keeping their old memories along with gaining new memories.
