    After clicking the clothing option Jeff scrolled through the various options and came upon the oddest option it was title "Clothing Set Options". Curious Jeff clicked this option and found various other transformation options some of which were sport uniform, school uniform, military uniform or gear, police uniform or gear, swat uniform or gear, firefighter uniform or gear, ect.... Confused by what some of the options did, he clicked the help button and a window popped up displaying:

    Cloth Set Options Help:

    This option will change the person into an entire set of cloths. For example: The school uniform option will change the person into the entire set of cloths: shirt, tie, vest, pants, shoe, and ect. Some options have the "or gear” extension this is for if the person wants to become the other nonclothing articles for a set or example if the person picks the military option there are not only clothing set for army, navy, national guard, marines, air force; there, are also gear set for the person to become such as if person selects the army and gear options that person would transform into all the things in the uniform that aren't considered clothing which would be things like weapons, night vision, binoculars, inferred, scopes, ect. All sets that have nonclothing things in the uniform sets have this option and both the clothing set and gear sets can be use make the person all the things in both sets.

    **Note1: You can also limit what you change into but you must change into 2 things either at least 1 from clothing set and at least 1 from gear set or 2 in either the clothing set or gear set.

    **Note2: And where they apply dress uniform, desk uniform, field uniform, and combat uniform subsets are available.

    **Note3: Person also gets to choose what rank or status in the set they are becoming will be if it applies for example: officer, rookie, private, sergeant, ect. And for military and police uniform and gear options things like camo type (desert, jungle, winter, ect) and uniform subclass (State police, Local police, K9 police) where they apply.

    **Note4: MP (Military Police) is located under police uniform option as a subset

    **Note5: Protective clothing such as shoulder pads, body armor, helmets, ect are included in the clothing sets

    *****Warning: All parts of clothing set, gear set or both gear and clothing set must be with in 100 yards of each other in order of the person transformed to change back and person will return to original transformation location and time.

    ****Note: If part of set person changed into is destroyed that part of set will not count for the above condition. If all parts of the set are destroy then the person will be changed back to as they were before the transformation.
