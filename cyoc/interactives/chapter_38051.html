    A brand new cell phone company is giving out promotional phones to some people. These phones aren't your typical cell phones though. These phones can transform anyone or any animal. The only condition is that it has to be alive, or just recently dead due to aging by the phone.

    An instruction manual comes with the phone. At the very beginning of the manual is the instructions on how to use the phone for transformations.

    This is your new TransFone, a cell phone capable of transforming anyone of any animal into anything. You can use it as a regular cell phone through your provider (See later chapters on how to use the phone in Phone Mode). You can switch to TransFone Mode by pressing *#*0000#*. The phone does not need to be turned on to do this. Once you have finished putting in the pass code you will be asked to enter a new four digit pass code. You will then be taken to the TransFone screen.

    On the TransFone screen you will see six choices. They are:

    Age: |----------| 01

    This is the age setting. Age can be set anywhere from 1 year old to 99 years old. Warning, if you age the subject (person or animal that is being transformed) past the age they would normally of died of natural causes there is a chance that they will die within a few seconds. If this happens you only have a minute to either Undo the transformation or regress them.

    Gender: M S F

    This is the Gender setting. The settings are for (M)ale,
    (S)he-male, and (F)emale. Depending on which one is selected the subject will transform into a male, she-male (girl with a penis), or female unless they are already the selected gender.

    Alter Reality: Y N

    This is the Alter Reality setting. Setting this to (Y)es will alter reality based on the age and gender selected. Setting this to (N)o will not alter reality. If reality is altered then everyone's memories will change except for the subject and/or user (the one who is using the phone) based upon whether or not the following two settings are set to yes.

    Aware: Y N

    This is the Aware setting. Setting this to (Y)es will allow the subject to retain his/her memories. Setting this to (N)o will wipe the subject's memories and replace them with memories of the altered reality. This setting can only be selected if yes is selected on the Alter Reality setting.

    User Aware: Y N

    This is the User Aware Setting. It is the same as the Aware setting except that it pertains to the user of the phone instead of the subject. This setting can only be selected if yes is selected on the Alter Reality setting.

    Memory Pri: N O

    This is the Memory Priority setting. Setting this to (N)ew will give priority to the new memories of the altered reality. The old memories will still be there but will seem like a dream. Setting this to (O)ld will give the old memories priority and the new memories will seem like a dream. This setting can only be selected if the Aware and/or User Aware settings are set to yes.

    The navigation key is used to select and change these settings.

    You will also notice Reg/Pkg and Undo in the lower corners of the screen. The button below Reg/Pkg switches between Regular and Package functions. The button below Undo reverses any change made by any TransFone.

    The green send button is used to initiate the transformation.

    The red end/power button is used to exit TransFone Mode.

    The numeric keypad is used with the package function. You can store up to ten packages in the phone. You can either create your own packages on your computer or download them from the TransFone website and transfer them to one of the ten numeric keys with the included cable. The keypad can only be used with the package function. When using the package function the screen will display the current package. To change the package just press the corresponding number where you stored the package.

    The * and # keys have no function.

    TransFone will change anyone or any animal that you point it at and has a range of 50 feet.

    TransFone Mode will not drain the phone's battery. This mode uses Zero Point Energy.
