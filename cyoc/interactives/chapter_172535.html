    The Age Change Pocket Watch

    Previous chapter
    Branch Full view
    added 4 years ago O

    This mysterious item is the Age Change Pocket Watch. It can turn backwards and forwards. This item can perform one of 6 functions.

    Self-Regression:
    Can target himself/herself to reverse it's age. Ether going fast or slow. When reaches into newborn age, the Self-Regression function will deactivate at will and won't go any further. Can use Self-Progression function to return to normal.

    Self-Progression:
    Can target himself/herself to advance it's age. Ether going fast or slow. Target will not contract any diseases associated with old age if they use this to become old. When him/her reaches 100 years of age, the Self-Progression function will deactivate at will and won't go any further. Can use Self-Regression function to return to normal.

    Family Regression:
    Can target a family to reverse it's age. It can be also used for Married couples to create a link to Reverse their age too. When him/her reaches into newborn age, the Self-Regression function will deactivate at will and won't go any further, but a Mother or Father will continue using this function. Can use Family-Progression function to return to normal.

    Family Progression:
    Can target a family to advance it's age. It can be also used for Married couples to create a link to advance their age too. Targets will not contract any diseases associated with old age if they use this to become old. When him/her reaches 100 years of age, the Self-Progression function will deactivate at will and won't go any further. Can use Family-Regression function to return to normal.

    Time Regression:
    Be a time traveler and use it to reverse time. Ether going fast or slow. He/She will see the experience of a timeline or history going backwards. When Self-Regression or Family Regression activates, Himself/Herself will go backwards with it. Can use Time-Progression function to return time to normal.

    Time Progression:
    Be a time traveler and use it to affect time. He/She will see the experience of a timeline speeding up. When Self-Progression or Family Progression activates, Himself/Herself will speed up with it. Can use Time-Regression function to return time to normal.

    With all of the functions prepared, who will be brought first?
