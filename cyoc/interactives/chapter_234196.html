    Chronivac Version 4.0

    An annoying digital cat becomes real

    added by PoofSkunk A month ago O

    No sooner than the moment the Antivirus was deleted did FluffyFifi make its appearance. Near the middle of the screen, the sprite of a white cat with a mischievous Cheshire grin popped into existence. The sprite opened its mouth and a piercing chip-tuned melody played over the computer speakers. It then vanished before popping up seconds later in a different corner and doing it once again.

    That would’ve been all it ever did if it hadn’t come across the Chronivac. That strange program was not only designed not only to warp reality, but to adapt to whatever the user wanted. As the virus's code brushed against it, some of that code rubbed off on it. The programming languages of the two programs were different, and the tiny, little bit of code that had found itself attached to it did not contain the info needed to translate it; but it did have the ability to change things, and so it began to change that foreign language around it into their own. Theirs was a very special programming language, however…

    As Jeff clicked hurriedly, trying everything he could to remove the virus, he began to notice that every time FluffyFifi popped up, it was slightly different. When it first popped up, it was 16-bit and only moved its mouth; soon, it looked more and more realistic, and it sauntered across his screen like a real cat striding, with the realism in movement that could've only been achieved by a master animator.

    The virus was now completely composed of that strange, reality-warping code. Its meagre programming began to develop into a brain, and that brain was suddenly filled with thoughts. Thoughts about what it could do now, as its new "computer language" wasn’t limited to computers.

    The cat was now no longer pixelated, and instead looked like a real cat, and Jeff's jaw dropped when suddenly the cat curiously batted at the screen with one of its paws, which passed through the screen three-dimensionally as if it wasn’t there.

    Suddenly, the cat jumped through the screen and into Jeff's lap. It looked up at him and grinned ear to ear. The cat somehow knew its own purpose. It was designed to be annoying - so annoying it would be.
